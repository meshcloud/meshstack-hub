How to use this template

  1. You can directly use this template in meshStack
OR
  1. You can copy the files in your own repository
  2. Go to meshStack and start the wizard to create a new building block definition
  3. Follow the instructions in the wizard and make sure you select "Terraform" as implementation type
  4. Add the link to your repository, you copied the files and connect it to meshStack
  5. meshStack will automatically add the inputs and outputs
  6. Add the end click Create Definition to finish
